
Spring Boot Login Api Using JWT Authentication Token




## API Reference

#### User Signin

```http
  POST /api/auth/login
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `Username` | `string` | **Required**. |
| `Password` | `string` | **Required**. |

#### User Signup

```http
  POST /api/auth/signup
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `Username` | `string` | **Required**. |
| `Password` | `string` | **Required**. |
| `Email` | `string` | **Required**. |
